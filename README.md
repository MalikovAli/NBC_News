NBC_News
Bu layihə istifadəçilərə gündəmdə olan ən son xəbərləri izləmək və onlarla bağlı ətraflı məlumat almaq imkanı verən xəbər platformasıdır. Layihənin əsas məqsədi istifadəçilərə rahat və interaktiv interfeys vasitəsilə aktual xəbərləri təqdim etməkdir.

💻 Texnologiyalar
HTML

CSS

React

✨ Xüsusiyyətlər
İstifadəçilər NBC_News platformasında aşağıdakı imkanlara sahibdirlər:

Xəbər İzləmə: Gündəmdə olan son xəbərlərə baxa və onlar haqqında ətraflı məlumat əldə edə bilərlər.

Ən Son Hekayələr: "Ən Son Hekayələr" bölməsindəki xəbərlərin üzərinə klikləməklə xəbərin tam təfərrüatlarını oxumaq mümkündür.

Davamını Gör: "Davamını Gör" düyməsi vasitəsilə siyahını genişləndirərək daha çox xəbərə çıxış əldə edə bilərsiniz.

Slider Sistemi: "Redaktorun Seçimləri" bölməsində slider vasitəsilə xəbərləri 2-2 olaraq sürüşdürərək izləyə bilərsiniz.

İnteraktivlik: Hər xəbər üçün save və like düymələri mövcuddur. Favoritlərə əlavə olunan xəbərlər UseBookmark.jsx faylında saxlanılır və idarə olunur.

🛠 Proses
Layihənin qurulma prosesi mərhələli şəkildə həyata keçirilib:

Dizayn və Struktur: İlk növbədə layihənin əsas vizual dizaynı və detail səhifəsinin strukturu quruldu.

Data İdarəçiliyi: Xəbər məlumatları ilkin mərhələdə statik və mock data olaraq hazırlandı.

Dinamik Xəbər Yükənməsi: "Ən Son Hekayələr" bölməsi üçün xüsusi düymə yerləşdirildi. Düyməyə klik edildikdə yeni xəbər blokları dinamik şəkildə ekrana gəlir.

Slider İnkişafı: "Redaktorun Seçimləri" bölməsi üçün xəbərlərin 2-2 qruplaşdırıldığı slider funksionallığı yazıldı.

Bookmark Sistemi: Favoritləmə funksiyası üçün xüsusi UseBookmark.jsx komponenti yaradıldı və dizayn olaraq inteqrasiya edildi.
